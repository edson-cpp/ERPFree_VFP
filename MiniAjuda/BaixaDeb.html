Procedimentos básicos.

Botões:

      Baixar Débitos

  1. Baixar – Clique aqui para baixar o(s) crédito(s) selecionado(s).
  2. Fechar – Fecha a tela Baixar Débitos.

Funções:

  1. Filtro por período: Exibe os débitos do período informado por data de vencimento.
  2. Filtro por situação: Exibe os débitos por situação, abertos e/ou baixados.
  3. Baixar Débito: Selecione um débito aberto na grade pressionando a Barra de Espaço, para desmarcar pressione a Barra de Espaço sobre um débito selecionado, clique em Baixar, pressione Enter ou dê Clique Duplo sobre o débito para baixá-lo com data atual e valor total do débito.
  4. Baixa Parcial: Selecione um débito aberto na grade pressionando a Barra de Espaço, para desmarcar pressione a Barra de Espaço sobre um débito selecionado, selecione o campo Data Pagto e informe a data do pagamento, selecione o campo Valor Pago e informe o valor que está sendo pago, clique em Baixar, informe os dados do novo débito na tela que irá se abrir e clique em Salvar.
  5. Baixar mais de um débito: Use Barra de Espaço para selecionar os débitos a serem baixados, use o procedimento 3 para baixa total e o procedimento 4 para baixa parcial.
  6. Os botões tem atalhos para agilizar os procedimentos. Ex. Botão Fechar, o F de Fechar é sublinhado, isso indica que esta tecla combinada com a tecla de Alt aciona este botão, isto funciona para todos os botões. Pressionar Alt+F simultaneamente é o mesmo que clicar no botão Fechar.
