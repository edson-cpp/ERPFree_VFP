Procedimentos básicos.

Botões:

      Lançamento de Cheques

  1. Salvar – Grava as informações do registro.
  2. Limpar – Limpa a tela para iniciar a digitação de outro cheque.
  3. Fechar – Fecha a tela Lançamento de Cheques.

Funções:

  1. No campo código do produto pressione F5 ou Clique Duplo no mouse para localizar por descrição.
  2. Para habilitar o campo Bom Para marque a caixa de seleção á esquerda do campo como Pré Datado.
  3. Para informar emitente selecione a caixa de checagem Emitido por terceiro. Informe o emitente somente em caso de emissão por terceiros, se o cheque for emitido pelo cliente não há necessidade de informar emitente.
  4. Os botões tem atalhos para agilizar os procedimentos. Ex. Botão Fechar, o F de Fechar é sublinhado, isso indica que esta tecla combinada com a tecla de Alt aciona este botão, isto funciona para todos os botões. Pressionar Alt+F simultaneamente é o mesmo que clicar no botão Fechar.
