Procedimentos básicos.

Botões:

      Cadastro de observações da nota fiscal

  1. Novo/Salvar – Cria um novo cadastro. / Grava as informações do registro.
  2. Alterar/Cancelar – Habilita os campos para alterações no registro selecionado. / Cancela o cadastro corrente.
  3. Localizar – Localiza a partir da lista de registros.
  4. Excluir – Exclui o registro selecionado.
  5. Primeiro( |< ) – Navega para o primeiro registro.
  6. Anterior( < ) – Navega para o registro anterior.
  7. Próximo( > ) – Navega para o próximo registro.
  8. Último( >| ) – Navega para o último registro.
  9. Fechar – Fecha a tela de cadastro.

Funções:

  1. No campo código do grupo pressione F5 ou Clique Duplo no mouse para localizar por descrição.
  2. Ctrl + Home – Navega para o primeiro registro.
  3. Ctrl + Page Up – Navega para o registro anterior.
  4. Ctrl + Page Down – Navega para o próximo registro.
  5. Ctrl + End – Navega para o último registro.
  6. Os botões tem atalhos para agilizar os procedimentos. Ex. Botão Fechar, o F de Fechar é sublinhado, isso indica que esta tecla combinada com a tecla de Alt aciona este botão, isto funciona para todos os botões. Pressionar Alt+F simultaneamente é o mesmo que clicar no botão Fechar.
