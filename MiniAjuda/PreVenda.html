Procedimentos básicos.

Botões:

      Pré-Venda / Orçamento

  1. Confirmar – Finaliza a pré-venda ou orçamento conforme selecionado no campo Tipo.
  2. Cancelar – Cancela operação atual.
  3. Alterar – Seleciona a grade de produtos permitindo a exclusão ou diminuição na quantidade de um item que já tenha sido inserido.
  4. Fechar – Fecha a tela Pré-Venda / Orçamento.

Funções:

  1. No campo código do cliente pressione F5 ou Clique Duplo no mouse para localizar por nome.
  2. No campo código do vendedor pressione F5 ou Clique Duplo no mouse para localizar por nome.
  3. No campo código do produto pressione F5 ou Clique Duplo no mouse para localizar por descrição.
  4. A tecla F9 permite conceder desconto na pré-venda (o desconto é concedido mediante autenticação – senha – de pessoa autorizada)
  5. Na grade de produtos (depois de pressionado o botão Alterar) use - (menos) para subtrair 1 na quantidade do produto selecionado e use Del para excluí-lo da grade.
  6. Os botões tem atalhos para agilizar os procedimentos. Ex. Botão Fechar, o F de Fechar é sublinhado, isso indica que esta tecla combinada com a tecla de Alt aciona este botão, isto funciona para todos os botões. Pressionar Alt+F simultaneamente é o mesmo que clicar no botão Fechar.
